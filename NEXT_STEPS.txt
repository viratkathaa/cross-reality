STEP 3: BUILD FOR QUEST 3 (15-30 MINUTES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Connect your Quest 3 to PC via USB
  2. File > Build Settings
  3. Select Android platform
  4. Click "Switch Platform" (waits for compile)
  5. File > Build and Run
  6. Select your Quest 3 device
  7. Unity compiles and deploys (takes ~10-15 mins)
  8. App launches automatically on Quest

  1. Put on the Quest 3

WHAT IF SOMETHING BREAKS?
━━━━━━━━━━━━━━━━━━━━━

❌ "API Key not set"
   → Paste your key in Inspector (STEP 2)

❌ "Failed to capture screenshot"
   → Restart the app, ensure camera is enabled

❌ "API request failed"
   → Check internet connection, verify API key

❌ "Model doesn't appear"
   → Check console logs (Window > General > Console)
   → Verify you have FAL AI credits

❌ "Can't grab model"
   → Make sure both hands are visible
   → Try pinching above the model

❌ Build fails
   → Switch to Android platform (File > Build Settings)
   → Ensure Quest 3 is connected
